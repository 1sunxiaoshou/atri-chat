# 命令系统设计

---

## 一、设计原则

1. 所有对角色/场景的指令通过**命令对象**统一表达
2. 命令是**可序列化的、与来源无关的纯数据对象**
3. 业务层只需要 `dispatch(command)`，不关心实现细节
4. 命令分为不同优先级和路由策略

---

## 二、完整命令类型定义

```typescript
// vrm-director/commands.ts

// ============================================================
// 公共字段：所有命令可选 targetId（默认为主角色）
// ============================================================
// 多实体场景下，命令可以指定作用于哪个实体
// 未指定 targetId 时默认作用于 isMain=true 的主角色

// ============================================================
// 叙事命令（有序队列 FIFO 执行）
// ============================================================
type NarrativeCommand =
  | {
      type: 'NARRATE_SEGMENT';
      payload: {
        text: string;                // 纯文本（已去除标记）
        audioUrl?: string;           // TTS 音频 URL
        motionName?: string;         // 指定动作名 (reply分类)
        motionCategory?: 'reply' | 'idle' | 'thinking';
        emotion?: string;            // 情感表情名
        emotionIntensity?: number;   // 0~1
        waitForMotion?: boolean;     // 音频完成后是否等动作也完成
      };
    };

// ============================================================
// 流程命令（高优先级，立即打断队列）
// ============================================================
type FlowCommand =
  | { type: 'ENTER_IDLE' }
  | { type: 'ENTER_THINKING' }
  | { type: 'ENTER_LISTENING' }
  | { type: 'INTERRUPT' }           // 打断叙事，回到 idle
  | { type: 'STOP_ALL' }            // 立即停止一切
  | { type: 'WAIT'; ms: number }    // 叙事中的停顿（在叙事队列里排队）

// ============================================================
// 表情命令（立即，并行执行，不排队）
// ============================================================
type ExpressionCommand =
  | {
      type: 'SET_EMOTION';
      targetId?: string;            // 可选，默认主角色
      emotion: string;              // 'neutral'|'happy'|'sad'|...
      intensity?: number;           // 0~1，默认 1.0
      blendMs?: number;             // 渐变时间，默认 200ms
    };

// ============================================================
// 动作命令（叙事队列外的独立动作）
// ============================================================
type MotionCommand =
  | {
      type: 'PLAY_MOTION';
      targetId?: string;            // 可选，默认主角色
      motion: string;               // 动作名或 URL
      category?: 'initial' | 'idle' | 'thinking' | 'reply';
      loop?: boolean;
    };

// ============================================================
// 移动命令（复合原子操作：位移 + 行走动画）
// ============================================================
type MovementCommand =
  | {
      type: 'MOVE';
      targetId?: string;            // 可选，默认主角色
      target: [number, number, number]; // 目标位置
      speed?: number;               // 移动速度，默认 1.5
      gait?: 'walk' | 'run';       // 步态选择
    }
  | {
      type: 'STOP_MOVE';
      targetId?: string;
    };

// ============================================================
// 镜头命令（立即，独立通道）
// ============================================================
type CameraCommand =
  | {
      type: 'CAMERA_CUT';
      mode: 'ots' | 'front' | 'fpv' | 'wide';
      dist?: 'close' | 'mid' | 'far';
    }
  | {
      type: 'CAMERA_TRANSITION';
      mode: 'ots' | 'front' | 'fpv' | 'wide';
      durationMs: number;
    }
  | {
      type: 'CAMERA_MOVE';
      payload: {
        preset?: CameraPreset;
        target?: [number, number, number];
        distance?: number;
      };
    }
  | {
      type: 'CAMERA_FOV';
      payload: { fov: number; duration?: number };
    };

// ============================================================
// 注视命令（立即）
// ============================================================
type GazeCommand =
  | { type: 'SET_GAZE'; target: 'user' | 'reset' | string };

// ============================================================
// 场景命令
// ============================================================
type SceneCommand =
  | { type: 'SET_BACKGROUND'; payload: BackgroundConfig }
  | { type: 'SET_GROUND'; payload: Partial<GroundConfig> };

// ============================================================
// 实体管理命令（多模型支持）
// ============================================================
type EntityCommand =
  | {
      type: 'ADD_ENTITY';
      payload: {
        id: string;
        entityType: 'vrm' | 'prop' | 'static_gltf';
        url: string;
        position?: [number, number, number];
        scale?: number;
        visible?: boolean;
      };
    }
  | { type: 'REMOVE_ENTITY'; entityId: string }
  | { type: 'SHOW_ENTITY'; entityId: string }
  | { type: 'HIDE_ENTITY'; entityId: string }
  | {
      type: 'ATTACH_OBJECT';
      objectId: string;           // 要挂载的道具 ID
      parentId: string;           // 挂到哪个角色上
      bone: string;               // 挂载骨骼名 (如 'RightHand')
    }
  | { type: 'DETACH_OBJECT'; objectId: string };

// ============================================================
// 角色变换命令
// ============================================================
type CharacterCommand =
  | { type: 'LOAD_CHARACTER'; characterId: string; avatarUrl: string }
  | { type: 'SET_CHARACTER_POSITION'; targetId?: string; payload: { offset: [number, number, number] } }
  | { type: 'SET_CHARACTER_SCALE'; targetId?: string; payload: { scale: number } };

// ============================================================
// SFX 命令（立即，可叠加）
// ============================================================
type SFXCommand =
  | { type: 'PLAY_SFX'; url: string; volume?: number };

// ============================================================
// 总命令联合类型
// ============================================================
export type VRMCommand =
  | NarrativeCommand
  | FlowCommand
  | ExpressionCommand
  | MotionCommand
  | MovementCommand
  | CameraCommand
  | GazeCommand
  | SceneCommand
  | EntityCommand
  | CharacterCommand
  | SFXCommand;
```

## 四、命令优先级与路由规则

```
命令类型               优先级    路由策略
────────────────────────────────────────────────────────
STOP_ALL              🔴 最高   中断一切，清空所有队列
INTERRUPT             🔴 高     清空叙事队列，执行中断动画
ENTER_THINKING        🔴 高     清空叙事队列，切换交互状态
ENTER_LISTENING       🔴 高     切换交互状态
ENTER_IDLE            🔴 高     切换交互状态
SET_EMOTION           🟡 中     旁路直达 ExpressionController（不排队）
CAMERA_*              🟡 中     旁路直达 CameraController（不排队）
SET_GAZE              🟡 中     旁路直达 GazeController（不排队）
PLAY_SFX              🟡 中     旁路直达 SFXController（不排队）
SET_BACKGROUND        🟡 中     旁路直达 SceneController（不排队）
PLAY_MOTION           🟡 中     写入 Store.motion（不排队）
MOVE / STOP_MOVE      🟡 中     MotionBridge 联动行走动画 + useFrame 位移
ADD/REMOVE_ENTITY     🟡 中     写入 Store.entities（不排队）
ATTACH/DETACH_OBJECT  🟡 中     修改 Store.entities 的 parentId/boneName
SHOW/HIDE_ENTITY      🟡 中     修改 Store.entities.visible
NARRATE_SEGMENT       🟢 低     追加到叙事队列（FIFO）
WAIT                  🟢 低     追加到叙事队列（FIFO）
LOAD_CHARACTER        🟢 低     串行执行，等待完成
```

## 五、CommandRouter 实现

```typescript
// vrm-director/CommandRouter.ts

export class CommandRouter {
  constructor(
    private narrative: NarrativeController,
    private expression: ExpressionController,
    private motionBridge: MotionBridge,
    private camera: CameraController,
    private gaze: GazeController,
    private sfx: SFXController,
    private scene: SceneController,
    private engagementSM: EngagementStateMachine,
    private store: typeof useVRMWorldStore,
  ) {}

  route(command: VRMCommand): void {
    switch (command.type) {
      // ===== 🔴 高优先级：流程命令 =====
      case 'STOP_ALL':
        this.narrative.clear();
        this.engagementSM.send('STOP_ALL');
        break;
      case 'INTERRUPT':
        this.narrative.clear();
        this.engagementSM.send('INTERRUPT');
        break;
      case 'ENTER_THINKING':
        this.narrative.clear();
        this.engagementSM.send('USER_SUBMIT');
        break;
      case 'ENTER_LISTENING':
        this.engagementSM.send('USER_INPUT_START');
        break;
      case 'ENTER_IDLE':
        this.engagementSM.send('ENTER_IDLE');
        break;

      // ===== 🟡 中优先级：旁路立即执行 =====
      case 'SET_EMOTION':
        this.expression.set(command.emotion, command.intensity, command.blendMs);
        break;
      case 'CAMERA_CUT':
      case 'CAMERA_TRANSITION':
      case 'CAMERA_MOVE':
      case 'CAMERA_FOV':
        this.camera.execute(command);
        break;
      case 'SET_GAZE':
        this.gaze.set(command.target);
        break;
      case 'PLAY_SFX':
        this.sfx.play(command.url, command.volume);
        break;
      case 'SET_BACKGROUND':
      case 'SET_GROUND':
        this.scene.execute(command);
        break;
      case 'PLAY_MOTION':
        this.motionBridge.play(command.motion, command.category, command.loop);
        break;

      // ===== 🟢 低优先级：叙事队列 =====
      case 'NARRATE_SEGMENT':
        // 第一个 segment 触发状态机转换
        if (this.engagementSM.current !== 'narrating') {
          this.engagementSM.send('NARRATE_START');
        }
        this.narrative.push(command.payload);
        break;
      case 'WAIT':
        this.narrative.pushWait(command.ms);
        break;

      // ===== 角色/场景 =====
      case 'LOAD_CHARACTER':
        this.store.setState({
          character: { modelUrl: command.avatarUrl, isLoading: true, error: null }
        });
        break;
      case 'SET_CHARACTER_POSITION':
      case 'SET_CHARACTER_SCALE':
        this.scene.executeCharacter(command);
        break;
    }
  }
}
```

## 六、业务层使用方式

### 原来（紧耦合）
```typescript
// 在 useChat.ts 里
const vrm = useVRM(character, isVRMMode);
vrm.startThinking();   // 直接调用 Hook 的方法
vrm.playSegments(segments); // 直接调用
```

### 新架构（松耦合）
```typescript
// 在 useAgentStream 的 submitWithContext 里
const director = useVRMWorldStore(s => s.director);

// 用户发消息
director.dispatch({ type: 'ENTER_THINKING' });

// 收到 AI 的流式片段（通过 StreamAdapter）
director.dispatch({ type: 'NARRATE_SEGMENT', payload: segment });

// 需要停止
director.interrupt();
```

`useChat` / `useAgentStream` 完全不知道"thinking 动作怎么选"、"音频怎么播"，只管发命令。
