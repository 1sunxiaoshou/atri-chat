## 🚀 快速开始

### 新手入门
1. 阅读 [项目规则](../.kiro/steering/project-rules.md) 了解开发规范
2. 配置 [国际化](./i18n-scripts.md) 开发环境
3. 了解 [TTS 架构](./TTS架构.md) 和 [VRM 系统](./VRM使用指南.md)

### 开发者指南
- **前端开发**: TypeScript + React 19 + Vite + Tailwind CSS
- **后端开发**: Python + FastAPI + SQLite
- **3D 渲染**: Three.js + @pixiv/three-vrm (即将升级到 React Three Fiber)
- **国际化**: 基于 JSON 的翻译系统
