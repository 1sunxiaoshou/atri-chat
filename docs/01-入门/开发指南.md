# 开发指南

本文档是本地开发入口。系统设计见 [系统设计](../02-架构/系统设计.md)，发布流程见 [桌面构建与发布指南](./桌面构建与发布指南.md)。

## 环境要求

- Python 3.12+
- uv
- Node.js 18+
- Rust 工具链
- FFmpeg

## 安装依赖

```bash
uv sync
npm install --prefix frontend
```

## 启动开发环境

后端：

```bash
uv run main.py
```

前端：

```bash
npm --prefix frontend run dev
```

Tauri 桌面开发：

```bash
npm run tauri:dev
```

默认端口：

| 服务 | 端口 |
|---|---|
| FastAPI | `9099` |
| Vite | `9900` |

## 常用检查

后端：

```bash
uv run ruff check .
uv run black --check .
uv run pytest
```

前端：

```bash
npm --prefix frontend run type-check
npm --prefix frontend run lint
npm --prefix frontend run check-i18n
```

命令系统解析检查：

```bash
npm --prefix frontend run check-commands
```

发布前检查：

```bash
uv run python scripts/release.py check
uv run python scripts/bump_version.py --check
```

## 主要目录

| 路径 | 用途 |
|---|---|
| `main.py` | FastAPI 入口 |
| `api/` | HTTP 路由与 schema |
| `core/` | 后端业务逻辑 |
| `core/agent/` | LangChain / LangGraph Agent |
| `core/db/` | ORM、数据库初始化、迁移 |
| `core/tools/` | Agent 工具 |
| `frontend/` | React 前端 |
| `frontend/src-tauri/` | Tauri 壳层 |
| `frontend/components/vrm/` | VRM 渲染与执行逻辑 |
| `scripts/` | 发布、版本、清理脚本 |
| `docs/` | 项目文档 |

## 开发约定

- 后端 API 变更时，同时检查前端调用。
- UI 文案变更时，同步更新 `frontend/locales/`。
- 运行时目录、端口、启动链路变更时，检查 Tauri 与 Python 两侧。
- `data/`、`build/`、`dist/`、`release_package/`、`frontend/dist/`、`frontend/node_modules/` 通常是生成物，不作为普通源码编辑。
- 文档事实写入 `docs/02-架构/`，未来计划写入 `docs/03-规划/`。
