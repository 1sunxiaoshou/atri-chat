## 技术实现

### LangChain Profile 集成

```python
# BaseProvider.get_model_info() 方法
def get_model_info(self, model_id: str, provider_config: ProviderConfig) -> ProviderModelInfo:
    """优先使用 LangChain profile 获取能力信息"""
    try:
        model_instance = self.create_text_model(model_id, provider_config)
        
        if hasattr(model_instance, 'profile') and model_instance.profile:
            profile = model_instance.profile
            
            # 从 profile 提取能力
            capabilities = []
            if profile.get('image_inputs'):
                capabilities.extend([ModelCapability.VISION, ModelCapability.DOCUMENT])
            if profile.get('audio_inputs') or profile.get('audio_outputs'):
                capabilities.append(ModelCapability.AUDIO)
            if profile.get('video_inputs'):
                capabilities.append(ModelCapability.VIDEO)
            if profile.get('tool_calling'):
                capabilities.append(ModelCapability.TOOL_USE)
            if profile.get('reasoning_output'):
                capabilities.append(ModelCapability.REASONING)
            
            return ProviderModelInfo(
                model_id=model_id,
                type=ModelType.CHAT,
                capabilities=capabilities,
                context_window=profile.get('max_input_tokens'),
                max_output=profile.get('max_output_tokens'),
            )
    except Exception as e:
        # 降级到名称推断
        pass
```

### 能力类型

系统支持以下模型能力：

- **VISION**: 图像输入支持
- **DOCUMENT**: 文档理解
- **AUDIO**: 音频输入/输出
- **VIDEO**: 视频输入
- **TOOL_USE**: 工具调用/函数调用
- **REASONING**: 深度推理/思考

### 通用参数

所有供应商支持以下通用参数：

- **temperature**: 温度 (0-2)
- **top_p**: 核采样 (0-1)
- **max_tokens**: 最大输出 token 数

### 供应商特定参数

不同供应商支持特定的高级参数：

| 供应商    | 参数名           | 类型      | 说明                                      |
| --------- | ---------------- | --------- | ----------------------------------------- |
| OpenAI    | reasoning_effort | segmented | 推理强度：低/中/高                        |
| Anthropic | thinking         | object    | 思考配置：type, budgetTokens              |
| Google    | thinking_config  | object    | 思考配置：thinkingBudget, includeThoughts |
| Tongyi    | enable_thinking  | boolean   | 启用深度思考                              |

## 优势

1. **自动化**: 使用 LangChain profile 自动识别能力，无需手动维护
2. **准确性**: 能力信息来自官方 models.dev 项目
3. **可扩展**: 新增供应商只需实现 BaseProvider 接口
4. **统一接口**: 所有供应商使用相同的参数管理方式
5. **降级策略**: profile 不可用时自动降级到名称推断
